**Punisher AI**

Punisher is a AI assistant built using  
React, Node.js, Web Speech API, and AI APIs

-> Features
- Voice input (speech-to-text)
- Customizable AI voice output
- Intelligent conversation (life, study, planning)
- Jarvis-style futuristic UI
- Voice control panel (rate, pitch, voice selection)

-> Tech Stack
- React (Vite)
- Node.js + Express
- Web Speech API
- AI API (OpenAI or alternative)

-> How to Run Locally

-> Backend
-> Terminal

cd backend
npm install
node index.js

-> Create .env file 

OPENAI_API_KEY=your_api_key_here

-> Frontend 
cd frontend
npm install
npm run dev
